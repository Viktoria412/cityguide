CityGuide — мобильное приложение-гид
Android-приложение для просмотра достопримечательностей города с картой, профилем пользователя и избранным.
Функциональность: главный экран, список достопримечательностей, карта (Google Maps), профиль с регистрацией/входом и выбором фото.
Технологии: Java, Android SDK 34, Fragment + BottomNavigationView, Google Maps SDK, Retrofit 2, Room Database, Glide, ViewModel + LiveData.
Структура проекта — дерево папок с файлами.
Как открыть: инструкция — склонировать, открыть в Android Studio, вставить свой Maps API ключ, запустить.
Минимальные требования: Android 7.0+, Google Play Services.
